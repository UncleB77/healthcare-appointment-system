# Healthcare Appointment Booking System

A web application for booking clinic appointments, featuring a responsive form with JavaScript validation and localStorage for saving bookings. Built for healthcare businesses needing efficient online scheduling.

## Features
- Responsive form with real-time validation for name, email, doctor, and date.
- Saves appointments using localStorage for persistent data.
- Displays booked appointments in a clean, user-friendly list.
- Mobile-friendly design with CSS styling.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Storage**: localStorage
- **Tools**: VS Code, Git, GitHub Pages

## Setup
1. Clone the repository: `git clone https://github.com/UncleB77/healthcare-appointment-system.git`
2. Open `index.html` in a web browser.
3. Host on GitHub Pages or Netlify for a live demo.
